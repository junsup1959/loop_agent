# Planning Gate Profile

## Planning Standard

- Decompose the approved goal into bounded work items and immutable revisions.
- Define ownership, dependencies, source and generated write scopes, base OIDs, outputs, context budgets, required professional references, and gates.
- Keep one Goal/run bound to one target repository while allowing independent branches and worktrees.
- Identify integration order, recovery ownership, and release-blocking conditions before execution.

## Decision Rule

Approve the plan only when every revision has one accountable owner, non-ambiguous scope, verifiable completion evidence, and an explicit path through required independent gates.
