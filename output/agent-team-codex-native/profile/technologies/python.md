# Python Technology Profile

## Discovery

Confirm supported Python versions, entry points, package metadata, dependency locks, frameworks, and execution environments from exact-OID manifests.

## Engineering Focus

- Keep imports directional and avoid import-time state mutation.
- Make exception, encoding, filesystem, subprocess, SQLite, and serialization behavior explicit.
- Define thread, process, asynchronous task, cancellation, and shutdown lifecycles.
- Preserve packaging, executable entry-point, and previous-version behavior.
- Use deterministic fixtures and isolate mutable local state.

## Validation

Run the repository's narrow tests and configured formatting, lint, type, packaging, or integration checks. Include one failure path and one affected I/O or state boundary.
