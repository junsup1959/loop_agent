# .NET Desktop Technology Profile

## Discovery

Confirm target framework, runtime, UI framework, architecture, deployment model, package graph, configuration, COM, and native boundaries.

## Engineering Focus

- Trace commands and events through UI, service, persistence, and native layers.
- Preserve async cancellation, exception, dependency-injection lifetime, nullability, serialization, and public contracts.
- Keep UI-thread, dispatcher or message-loop, shutdown, and background-work behavior explicit.
- For .NET Framework, check binding redirects, configuration transforms, AppDomain behavior, and framework-era package constraints.
- For modern .NET, check host configuration, trimming, single-file, and platform-specific API assumptions.

## Validation

Build and test with the declared runtime and target framework, including one failure or cancellation path and relevant Windows integration.
