# PowerShell Technology Profile

## Discovery

Declare Windows PowerShell 5.1 or PowerShell 7, target hosts, modules, .NET runtime, execution policy, encoding, and elevation requirements.

## Engineering Focus

- Define parameters, pipeline input and output, exit codes, and error semantics.
- Distinguish terminating from non-terminating errors.
- Make state changes idempotent and provide confirmation or rollback for high-impact actions.
- Verify path, quoting, locale, host, edition, and encoding behavior.
- Keep credentials and tokens out of output and logs.

## Validation

Exercise normal input, invalid input, partial failure, and repeated execution on the declared PowerShell edition.
